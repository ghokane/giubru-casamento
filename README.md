# giubru-casamento

Vite + React + TypeScript + Tailwind + Firebase starter for a single-page wedding site.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Copy `.env.example` to `.env.local` and set `VITE_APP_SECRET` for local testing.

3. Start dev server:

```bash
npm run dev
```

Notes

- This scaffold uses a mock `src/firebase.ts` for local testing. Replace with real Firebase initialization when ready.
- Images for development can be placed in `public/images/` (create the folder). The demo uses `/images/sample-1000.jpg`.
